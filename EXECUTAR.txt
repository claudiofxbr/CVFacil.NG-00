📖 RESUMO FINAL - COMO EXECUTAR CVFacil.NG
===============================================

🎯 3 FORMAS DE EXECUTAR (Do Mais Fácil ao Mais Controle)

┌─────────────────────────────────────────────────────────────────────┐
│ 1️⃣ FORMA MAIS FÁCIL - Script Automático ⭐                        │
└─────────────────────────────────────────────────────────────────────┘

WINDOWS:
  • Double-click em: start.bat

LINUX/MAC:
  chmod +x start.sh
  ./start.sh

✅ Resultado: Servidor inicia e navegador abre em http://localhost:3000


┌─────────────────────────────────────────────────────────────────────┐
│ 2️⃣ FORMA MANUAL - Comandos (Recomendado)                          │
└─────────────────────────────────────────────────────────────────────┘

# 1️⃣ Abrir terminal na pasta do projeto
cd C:\Users\VeKTI-01\Downloads\cvfacil-ng-dev

# 2️⃣ Instalar dependências (primeira vez)
npm install

# 3️⃣ Iniciar servidor
npm run dev

# 4️⃣ Abrir navegador
# Digite: http://localhost:3000

Tempo estimado: 5-10 minutos (primeira vez)


┌─────────────────────────────────────────────────────────────────────┐
│ 3️⃣ FORMA DEVELOPMENT - VS Code                                    │
└─────────────────────────────────────────────────────────────────────┘

1. Abrir projeto em VS Code
2. Terminal → New Terminal (Ctrl + ')
3. Executar: npm install + npm run dev
4. Ctrl + Click no link


✅ PRÉ-REQUISITOS (Verificar)
═════════════════════════════════

✓ Node.js v20+ (https://nodejs.org/)
  node --version

✓ npm v10+ (vem com Node.js)
  npm --version

✓ Arquivo .env.local deve existir
  ls .env.local  # ou dir .env.local no Windows

⚠️ ERROS COMUNS & SOLUÇÕES
══════════════════════════

──────────────────────────────┼─────────────────────────────
